# Lumina Beauty App

This is the full-stack version of the Lumina Beauty application.

## Structure
- **server.js**: The Node.js/Express backend that serves the app and provides API endpoints.
- **public/**: Contains the frontend code (HTML, CSS, JS).
- **package.json**: Project dependencies and scripts.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

## Running the App 
1. Start the server: 
   ```bash
   npm start
   ```
2. Open your browser to `http://localhost:3000`

## Deployment
Upload the following files/folders to your hosting provider (e.g., Heroku, Render, DigitalOcean):
- `package.json`
- `server.js`
- `public/`
